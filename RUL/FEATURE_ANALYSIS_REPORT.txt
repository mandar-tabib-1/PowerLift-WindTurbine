1. WIND TURBINE OPERATION OVERVIEW
================================================================================

Modern wind turbines are complex systems with hundreds of sensors monitoring:

MECHANICAL SYSTEMS:
├─ Gearbox (90:1 ratio) - Gearbox temperature, oil temperature, vibration
├─ Generator - Temperature, current, power output
├─ Rotor & Blades - Pitch angle, rotor speed, stress
└─ Nacelle Structure - Temperature, structural vibration

ELECTRICAL SYSTEMS:
├─ Power Electronics - Converter temperature, efficiency
├─ Grid Connection - Frequency, voltage, reactive power
└─ Control Systems - PLC, pitch controller, yaw controller

ENVIRONMENTAL SENSORS:
├─ Wind Speed & Direction
├─ Ambient Temperature
├─ Humidity
└─ Pressure

2. FAILURE MODES IN WIND TURBINES (Domain Expert Knowledge)
================================================================================

PRIMARY FAILURE MODE: Gearbox Bearing Failure
┌─────────────────────────────────────────────────────────────┐
│ Progression:                                                 │
│ 1. Normal operation (0-6000h)                               │
│    └─ Smooth bearing surface, low friction                  │
│ 2. Initial pitting (6000-7000h)                            │
│    └─ Microscopic surface damage, temp increases 5-10°C    │
│ 3. Spalling (7000-7500h)                                    │
│    └─ Larger cracks, vibration increases 2-3x               │
│ 4. Catastrophic failure (7500-8000h)                       │
│    └─ Bearing seizure, power loss, system shutdown         │
│                                                              │
│ KEY INDICATORS:                                              │
│ • Gearbox Temperature ↑ (exponential growth near failure)   │
│ • Vibration Level ↑ (early warning 200-300h before)         │
│ • Power Output ↓ (increased friction)                       │
│ • Oil Temperature ↑ (lubrication degradation)               │
│ • Generator Current ↑ (higher mechanical load)              │
└─────────────────────────────────────────────────────────────┘

SECONDARY FAILURE MODES:
1. Generator Bearing Failure - Similar to gearbox, different timeline
2. Blade Pitch Control Failure - Mechanical linkage issues
3. Power Electronics - IGBT failures in converters
4. Structural - Fatigue in tower/nacelle

3. CRITICAL SCADA FEATURES FOR RUL PREDICTION
================================================================================

FEATURE #1: GEARBOX TEMPERATURE
─────────────────────────────────
Purpose:        Direct indicator of bearing health
Units:          Celsius (°C)
Sensor Type:    Pt100 RTD thermometer
Healthy Range:  45-70°C
Degrading:      70-90°C
Critical:       90°C+
Failure Point:  >100°C (typically 110-120°C for shutdown)

Physics:
├─ Normal bearings: friction heat = 55°C + ambient
├─ Worn bearings: friction heat increases exponentially
├─ Formula: Heat ∝ μ × N × (1 + e^damage)
│   where μ = friction coefficient, N = bearing load, e = error
└─ Result: 10°C rise ≈ bearing damage level 30-50%

Data Characteristics:
├─ Steady state: Low variance (σ < 2°C)
├─ Degradation phase: Linear rise at 0.01°C/hour
├─ Failure phase: Exponential (0.1°C/hour)
├─ Noise level: 2-3°C (sensor + environment)
└─ Seasonality: Moderate (summer 5-10°C higher)

IMPORTANCE SCORE: 95/100 - CRITICAL FOR PREDICTION


FEATURE #2: VIBRATION LEVEL
────────────────────────────
Purpose:        Mechanical degradation (early warning)
Units:          mm/s (RMS velocity)
Sensor Type:    Accelerometer + integration
Healthy Range:  2-4 mm/s
Degrading:      4-8 mm/s
Critical:       8+ mm/s
Shutdown:       >12 mm/s

Physics:
├─ Loose bearings cause increased vibration
├─ Vibration ∝ bearing clearance × load
├─ Equation: V = A × f² (velocity = amplitude × frequency²)
└─ Detection: Effective 200-500 hours BEFORE temperature rise

Data Characteristics:
├─ Steady state: Low, periodic (wind-dependent)
├─ Degradation: Gradual 5-10% increase per month
├─ Failure: Sudden 3-5x spike (within 50 hours)
├─ Noise level: 0.3-0.5 mm/s (measurement noise)
├─ Frequency content: Bearing fault frequencies 10-20 kHz
└─ Variability: Wind speed dependent (3 mm/s variation with wind)

IMPORTANCE SCORE: 98/100 - MOST SENSITIVE EARLY INDICATOR


FEATURE #3: GEARBOX OIL TEMPERATURE
────────────────────────────────────
Purpose:        Lubrication health, bearing condition
Units:          Celsius (°C)
Sensor Type:    Pt100 RTD (submerged in oil sump)
Healthy Range:  50-65°C
Degrading:      65-85°C
Critical:       85°C+
Oil Viscosity:  50°C oil (ISO VG 46): ν=46 cSt
                At 85°C: ν=7.4 cSt (84% viscosity loss!)

Physics:
├─ Oil viscosity: ν(T) = K × e^(E_a/RT)
│   (Arrhenius equation for viscosity-temperature)
├─ Bearing friction: f = μ × N
│   where μ = viscosity-dependent friction coefficient
├─ Temperature rise: ΔT = (f × N × ω) / (ρ × c × m)
│   where ω=angular velocity, ρ=density, c=heat capacity
└─ Result: 1°C rise = ~2% viscosity loss

Data Characteristics:
├─ Steady state: Stable, tracks ambient + friction
├─ Response time: 30-60 minutes behind actual bearing temp
├─ Baseline variation: ±3°C with wind conditions
├─ Degradation rate: 0.01-0.05°C/hour
├─ Hysteresis: Temperature lags during transients
└─ Relationship: Oil temp ≈ 0.9 × Gearbox temp + offset

IMPORTANCE SCORE: 92/100 - CONFIRMS BEARING DEGRADATION


FEATURE #4: GENERATOR TEMPERATURE
──────────────────────────────────
Purpose:        Overall thermal stress, system loading
Units:          Celsius (°C)
Sensor Type:    Winding temperature (RTD)
Healthy Range:  55-75°C
Degrading:      75-95°C
Critical:       95°C+
Design Max:     155°C (insulation class F)

Physics:
├─ Generator heat: P_loss = I²R + core_loss
│   where I = current, R = resistance
├─ Temperature: T = T_ambient + ΔT_rise
├─ Thermal model: τ = L/R (time constant 10-30 min)
└─ Relationship: Generator load ∝ Power output

Data Characteristics:
├─ Variability: Highly dependent on power output
├─ Daily cycle: Peak during high wind, low at night
├─ Correlation: 0.85 with mechanical power
├─ Noise: 2-3°C (sensor variation)
├─ Response: Fast (5-10 minutes to peak)
└─ Relationship: Gen_temp ≈ 50 + 0.15×Power + wind_effect

IMPORTANCE SCORE: 72/100 - INDICATES STRESS LEVEL


FEATURE #5: GENERATOR POWER OUTPUT
───────────────────────────────────
Purpose:        Mechanical efficiency, load on system
Units:          kW
Sensor Type:    Power transducer (grid measurement)
Healthy Range:  1500-2500 kW (depends on wind speed)
Expected Curve: Power ∝ Wind³ (cubic law)
Efficiency:     80-95% of Betz limit

Physics:
├─ Power: P = 0.5 × ρ × A × Cp × v³
│   where v = wind speed, A = rotor area, Cp = 0.4-0.5
├─ Gearbox loss: 3% in healthy state, 5-10% when degraded
├─ Generator loss: 2-3% nominal, increases with load
└─ Expected vs actual: Deviation indicates mechanical problem

Data Characteristics:
├─ Variability: High (wind dependent)
├─ Expected value: Lookup table based on wind speed
├─ Deviation: Normal ±5%, >10% indicates problem
├─ Response: Fast (seconds to minutes)
├─ Correlation: Strong with wind speed (r=0.92)
├─ Outliers: Grid disconnections, pitch control events
└─ Trend: Slow degradation -1% per 1000 operating hours

IMPORTANCE SCORE: 78/100 - CONFIRMS MECHANICAL PROBLEMS

8. PREPROCESSING & NORMALIZATION
================================================================================

METHOD: StandardScaler (zero-mean, unit-variance)

Why StandardScaler?
├─ Autoencoder requires normalized input [-1, 1]
├─ Prevents large-magnitude features from dominating
├─ Improves LSTM convergence
└─ Necessary for GMM clustering

Formula:
    X_scaled = (X - mean) / std

Applied separately to:
├─ Training data: Fit scaler on X_train, transform X_train
├─ Test data: Transform only (using training parameters)
└─ New data: Transform using saved scaler
